# 微信电商小程序

#### 介绍
基于小程序原生MINA框架开发的电商小程序项目的个人练习demo
拥有vue框架知识上手该小程序学习会更顺畅

#### 软件架构
原生MINA框架、小程序原生API开发
最新免费数据接口文档：https://www.it120.cc/
该小程序使用的接口文档：https://www.showdoc.com.cn/128719739414963/2513235043485226

#### 注意事项

1.  该个人小程序项目不能实现支付功能
2.  功能未完全完善，仅作为练习使用
